딥러닝 기반 부실기업 예측모형에 관한 연구

DataSet

- 데이터 수집 가능 여부 : 수집 가능
- 대상기업 선정 : kospi, kosdaq, konex 상장기업 및 외부감사 대상기업 33,916개, 비금융, 결산월 12월
- 대상기간 : 2017 ~ 2019
- 데이터분할 : 70 대 30 비율로 학습, 정규화


데이터 전처리과정

결측치 중앙값 대체,  영업이익이자보상비율 컬럼 결측치 부분 해당 기업 데이터 삭제
이상치 윈저라이징 적용하여 처리 (0.01, 0.99)
 -> 최종 정상기업(13818) 한계기업(2995)


EDA 확인

- 기초통계량
- 상관분석
- 다중공선성

Feature

- 선정 및 다양성 여부
종속변수 : 설정한 한계기업, 독립변수 : 각 기업의 재무비율
성장성, 수익성, 활동성, 안정성 각 지표별 독립 표본 t 검정 진행 후 p-value가 0.05보다 큰 연도별 재무비율 제거
-> 연도별 재무비율로 로지스틱 회귀분석 실시 -> 후진선택법
*  후진선택법 : 잠정적인 독립변수로 선정된 전체 변수군을 대상으로 독립변수군에서 제외하여도 종속변수에 대한 설명력에 영향을 미치지 않는 변수들을 순차적으로 제거하는 방법을 의미
=> 총 2017년 11개, 2018년 12개, 2019년 11개 변수 선택

모델링

머신러닝
앙상블 모델 사용 
분류기 결합 방식 : 배깅 사용
구성 분류기 : 의사결정 나무, SVM, K-최근접 이웃법
의사결정나무 + 배깅 = 랜덤포레스트
=> 최종 랜덤포레스트, 서포트 벡터 머신,  K-최근접이웃법

딥러닝

RNN - LSTM, RNN - GRU


성능 결과

혼동행렬 지표 사용 (정밀도, 재현율, F1- SCORE)
(RF)를 제외한 다른 머신러닝 앙상블 모형에 비해 딥러닝 모형들이 우수한 예측 성능 보임
특히 재현율(RECALL) 부분에서 뛰어난 상황 보임 - 부실기업은 이 점수가 중요함


기여효과
딥러닝과 머신러닝을 비교하면서 딥러닝 또한 부실기업 예측에 있어 성능이 좋다는 결과를 얻음


한계점
- 저자 의견
재무비율만 사용 - 기업 외적인 변수 사용 X
회계정보 사용 - 기업의 주가와 같은 시장정보가 가지는 적시성, 미래지향성 부족
상장기업 같은 규모가 큰 기업만 데이터 사용 - 중소기업 X


- 발표조 의견
한계기업 정의를 한국은행의 '3년 연속 이자보상배율이 1미만인 기업' 만 사용해서 분석한 점
